# Enum in Java

## Definition

Enum is a special class used to store fixed constants.

---

# Example

```java
enum Day {

    MONDAY,
    TUESDAY
}
```

---

# Important Points

1. Stores constants.
2. Improves readability.
3. Used with switch.
4. Can have constructor.
5. Enum is type-safe.
