# Anonymous Class in Java

## Definition

An anonymous class is a class without name.

Used for:
- One-time object creation
- Short implementation

---

# Syntax

```java
new ClassName() {

};
```

---

# Important Points

1. No class name.
2. Used once.
3. Short code.
4. Common with abstract class and interface.
5. Improves readability.
