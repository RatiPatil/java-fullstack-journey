# Variables in Java

## Definition
Variables are containers used to store data in memory during program execution.

A variable allows the program to store, update, and manipulate values.

---

# Types of Variables in Java

| Variable Type | Description |
|---|---|
| Local Variable | Declared inside methods |
| Instance Variable | Declared inside class but outside methods |
| Static Variable | Shared among all objects using `static` keyword |
