# Literals in Java

## Definition
Literals are fixed values written directly in the source code.

They represent constant values assigned to variables.

---

# Types of Literals

| Literal Type | Example |
|---|---|
| Integer Literal | `10` |
| Floating Literal | `10.5` |
| Character Literal | `'A'` |
| String Literal | `"Java"` |
| Boolean Literal | `true` |
| Binary Literal | `0b1010` |
| Octal Literal | `012` |
| Hexadecimal Literal | `0x1A` |

---

# Example

```java
int number = 10;
char grade = 'A';
String name = "Java";
