# Serialization and Deserialization

## Serialization

Converting object into file/byte stream.

---

# Deserialization

Converting file/byte stream back into object.

---

# Important Points

1. Uses Serializable interface.
2. Saves object state.
3. Used in networking.
4. Used in file storage.
5. Object streams are used.

---

# Quick Revision

- Object → File = Serialization
- File → Object = Deserialization
