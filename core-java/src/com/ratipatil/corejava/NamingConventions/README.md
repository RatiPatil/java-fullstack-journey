# Java Naming Conventions

## Definition

Naming conventions are rules and standards used to give meaningful names to:
- Classes
- Variables
- Methods
- Packages
- Constants

Good naming improves:
- Readability
- Code understanding
- Professional coding style

---

# 1. Class Naming Convention

## Rule

- Class names should start with Capital Letter.
- Use PascalCase.

---

## Correct Examples

```java
Student
EmployeeDetails
BankAccount
```

---

## Wrong Examples

```java
student
employee_details
123Student
```

---

# Quick Revision

## Class Naming

- Start with capital letter
- Use PascalCase
- Meaningful names

---

# 2. Variable Naming Convention

## Rule

- Variable names should start with small letter.
- Use camelCase.

---

## Correct Examples

```java
studentName
studentAge
totalMarks
```

---

## Wrong Examples

```java
StudentName
student_name
123marks
```

---

# Quick Revision

## Variable Naming

- Start with small letter
- Use camelCase
- Meaningful names

---

# 3. Method Naming Convention

## Rule

- Method names should start with small letter.
- Use camelCase.
- Usually action words.

---

## Correct Examples

```java
displayStudentDetails()
calculateMarks()
showResult()
```

---

## Wrong Examples

```java
DisplayStudent()
student_details()
123show()
```

---

# Quick Revision

## Method Naming

- Start with small letter
- Use camelCase
- Action-based names

---

# 4. Package Naming Convention

## Rule

- Package names should always be lowercase.

---

## Correct Examples

```java
com.ratipatil.corejava
com.ratipatil.oops
```

---

## Wrong Examples

```java
CoreJava
COM.Java
```

---

# Quick Revision

## Package Naming

- Always lowercase
- Use domain style
- No spaces

---

# 5. Constant Naming Convention

## Rule

- Constants should use uppercase letters.
- Words separated using underscore `_`.

---

## Correct Examples

```java
MAX_VALUE
PI
COLLEGE_NAME
```

---

## Wrong Examples

```java
maxValue
Pi
collegeName
```

---

# Quick Revision

## Constant Naming

- All uppercase
- Use underscore `_`
- Fixed values

---

# Types of Naming Styles

| Style | Example |
|---|---|
| PascalCase | `StudentDetails` |
| camelCase | `studentName` |
| UPPER_CASE | `MAX_VALUE` |

---

# Simple Understanding

| Java Element | Naming Style |
|---|---|
| Class | PascalCase |
| Variable | camelCase |
| Method | camelCase |
| Package | lowercase |
| Constant | UPPER_CASE |
